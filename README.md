# Waitress rating 

## Overview
A mobile app that allows diners to quickly rate their server's performance at restaurants using a simple, streamlined rating system. The app focuses on providing immediate feedback to help improve service quality.

## Problem
Diners have no easy way to rate individual servers, and restaurants lack real-time feedback on service performance

## Target User
Restaurant customers who want to provide feedback on service quality

## Features
- Simple 1-5 star server rating
- Restaurant lookup by location
- Anonymous rating submission
- Basic server profile pages
- Rating history for users
